# springboot-learning-1
springboot框架相关技术的案例项目及使用springboot框架时的需注意点

### 项目介绍
SpringBoot整合框架各种实用的组件技术点，纯属个人技术积累和框架学习，有缺漏之处请指出。  

### 主体版本号
- java v1.8
- springboot v2.0.5.RELEASE
